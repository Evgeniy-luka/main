Исследование рынка предприятий общественного питания Москвы

Цель проекта:
Исследует рынок организаций общественного питания Москвы и сформулировать рекомендации для инновационного сложного проекта - кафе с роботизированным обслуживанием.

Описание проекта:
В ходе исследования проводится сравнение результатов общественного питания в Москве по различным характеристикам (форматам, сетям, количеству посадочных мест и т.д.), происходит декомпозиция адресов, и увеличивается количество данных из внешних источников информации о направлениях расположения точек общепита.

По результатам анализа подготовлена ​​презентация для снижения отклонений. При построении графиков использовались библиотеки Seaborn и Plotly.

Вывод исследования:
Самой популярной формой заведения общественного питания в Москве является кафе.
На первых порах создание обособленного кафе выглядит предпочтительнее - с нуля сети требуют огромных вложений, которые могут не окупиться, а нынешняя сеть вряд ли с развитием воспримут инновационную идею роботизированного обслуживания.
Если в перспективе всё же следить за развитием сети роботизированных кафе, то рациональнее всего выбрать формат фаст-фуда:
во-первых, из-за ограниченного ассортимента, приготавливаемого из полуфабрикатов, данный формат проще реализовать и контролировать в видеосети;
во-вторых, по тем же причинам, ограничивается программное обеспечение управления посетителями роботов, что приводит к снижению затрат на их разработку и производство.
Общее количество мест в пилотном заведении выбрать по среднему уровню для формирования кафе - 30 мест.
Рационально выбрать для размещения пилотного кафе одну из улиц с одним заведением общественного питания, расположенных недалеко от улиц, входящих в топ-10 по достаточно существенным результатам. Таким образом, за счет снижения проходимости может снизиться конкуренция, что будет поддерживать интерес клиентов в период становления и развития бизнеса.
Использование результатов исследования:
Результаты исследования предполагают использование инвестиций для принятия решений о создании нового заведения.

Технические особенности проекта:
В данных присутствуют неявные дубликаты - проведена предобработка.

В ходе исследований также потребовалось найти, загрузить и использовать данные из внешних источников, построить графики, диаграммы рассеяния, круговые диаграммы.
