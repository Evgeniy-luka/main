Анализ причин убытков компании-разработчика развлекательных приложений

Цель проекта:
Определить причину убытков компании в крупных рекламных вложениях и предложить пути разрешения данной ситуации.

Описание проекта:
Проведен исследовательский и значительный анализ данных маркетингового отдела компании (журнал сервера с данными о посещениях, выгрузка данных о покупках, данных о рекламных расходах). В ходе анализа рассчитываются различные метрики: LTV, CAC, ROI, CR, RR. На основании полученных результатов сформулированы рекомендации для отдела маркетинга.

Вывод исследования:
Причины убытков компании:
реклама в целом за период (2 недели) окупается на 80%;
виновниками недостаточной окупаемости являются три рекламных канала — TipTop, FaceBoom и AdNonSense;
Основной причиной недостаточной окупаемости рекламы в TipTop (более 50% общего рекламного бюджета) является регулярный опережающий рост ее стоимости;
недостаточная окупаемость рекламы в каналах FaceBoom (стабильно порядка 30% бюджета) и AdNonSense, обусловленная низким уровнем удержания пользователей (пловть до 0 в конце второй недели) - по прогнозам, вероятно, неправильно выбранной информации о ресурсах пользователей;
В разрезе устройства наблюдается недостаточное удержание пользователей платформ Mac, iPhone и Android.
Основные рекомендации для маркетингового отдела:
пересмотреть результаты работы эксперта по рынку;
Совместный пересмотр с излучателями FaceBoom и AdNonSense продвигает их рекламные стратегии, в частности, касательную ограничение блокировки мобильных приложений, что позволяет увеличить удержание клиентов по этим каналам и, с точки зрения теории, их окупаемость.
Использование результатов исследования:
Результаты исследования предполагается использовать при распределении рекламного бюджета между условиями.

Технические особенности проекта:
