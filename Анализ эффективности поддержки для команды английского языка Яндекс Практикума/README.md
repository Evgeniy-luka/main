Задача: рассмотреть динамику обращений в службы поддержки и оценить изменения основных показателей скорости работы между странами и маем в 2024 году. Что сделал:

Провел анализ двух массивов данных, рассчитал ключевые показатели: общее количество запросов, среднее время ответов и решений.
Выявил изменения в производительности, рассчитав разницу в средних значениях между периодами.
Исследовал запросы по каналам обращения и динамику времени первого ответа по каждому каналу.
Создал наглядные визуализации в Google Sheets для презентаций результатов вычислений. Инструменты: Google Sheets, статистический анализ (медиана).
