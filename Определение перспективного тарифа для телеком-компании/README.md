Определение перспективного тарифа для телеком-компании

Цель проекта:
На основе анализа тарифов телеком-компаний на небольшой выборке ее клиентов сделать статистически обоснованный вывод о более выгодных для компании тарифах.

Описание проекта:
В проекте проведён анализ использования клиентами тарифов, их поведения при использовании услуг оператора и сформулированы рекомендации по оптимальным наборам услуг для пользователей, статистические подтверждённые каналы проверки, полученные о различии вызовов абонентов разных тарифов и различии вызовов абонентов из Москвы и других регионов.

Вывод исследования:
Проверка статистических гипотез показала, что средняя выручка оператора от пользователей тарифных планов Ultra и Smart наблюдается причём плохо.
За счет большего количества абонентов тарифа Smart, а также за счет их склонности чаще перерасходовать более дорогие (дополнительная плата за превышение) услуг, можно утверждать, что именно тариф Smart более выгоден для Мегалайна по данным за 2018 год.
Использование результатов исследования:
Результаты исследования можно использовать для корректировки рекламного бюджета.

Технические особенности проекта:
Данные зашумлены пропусками, некорректными типами данных - проведена предобработка.

В ходе исследований также потребовалось обогатить данные, построить гистограммы, боксплоты.
