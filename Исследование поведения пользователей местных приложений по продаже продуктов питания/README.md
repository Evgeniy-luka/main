Исследование поведения пользователей местных приложений по продаже продуктов питания

Цели проект:
Определить, как пользователи доходят до покупок в приложении, сколько они доходят до покупок, а сколько — «застревает» на различных шагах воронки.
Проанализировать результаты A/A/B-эксперимента по замене шрифтов в приложении (будут ли они непривычны для пользователей и повлияют на их прохождение воронке).
Описание проекта:
Исследование построено на событийном анализе. В ходе него были построены воронки продаж, а также исследованы способы использования товаров. В процессе анализа результатов A/A/B-теста проведено сравнение 2 контрольных групп между собой, чтобы убедиться в правильности разделения трафика, а затем - сравнение с тестовой последовательностью. Установлено, что новый шрифт существенно не влияет на поведение пользователей.

Вывод исследования:
Данные за весь период времени признаны неполными. Неполные данные отброшены, в результате чего потеряно 1,16 % событий и менее 1 % пользователей.
В среднем на 1 пользователя произошло 32 события, среднее количество событий на пользователя равно 20.
Воронку сценария событий: MainScreenAppear, OffersScreenAppear, CartScreenAppear, PaymentScreenSuccessful.
Больше всего пользователей склоняется на шаг OffersScreenAppear (38,09 % от общего количества). До успешной оплаты добираются только 46,97 % пользователей - меньше половины!
При этом результаты А/А-теста для контрольных групп на заданном критическом уровне статистической революции 0,01 не основаны на количестве конверсий пользователей среди покупателей различных типов, а также не основаны на количестве конверсий для каждого шага воронки различных.
По результатам А/В-теста при заданном критическом уровне статистической инновации 0,01 можно утверждать, что изменение шрифта не привело к возникновению условия возникновения конверсии.
Использование результатов исследования:
Результаты исследования предполагают использование продуктов команды для разработки стратегии развития проб и улучшения опыта пользователей.

Технические особенности проекта:
В данных присутствуют явные дубликаты и некорректированные данные - проведена предобработка. В ходе исследований также требовалось построить гистограммы, диаграммы, визуализировать воронку событий.
