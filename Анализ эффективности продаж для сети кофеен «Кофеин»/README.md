Задача: выделить низкомаржинальные товары и учитывать ежемесячную динамику выручки для оптимизации ассортимента и управления запасами. Что сделал:

Провелзированный анализ продаж по товарным категориям и позициям.
Рассчитаны основные метрики для определения товаров с наименьшим вкладом в выручку.
Выявил 15% ассортимента, приносящего наименьшую выручку (<5% от общей), и подготовил рекомендации по их замене или оптимизации.
Построил помесячную динамику продаж с учетом сезонных факторов для прогнозирования солнца.
Автоматизировал отчетность с помощью интерактивных инструментов для регулярного Диптиха. Инструменты: Excel (сводные таблицы, формулы, визуализация).
