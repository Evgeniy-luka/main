2. Результаты решения ad hoc задачи
Существует ли зависимость активности игроков по совершению внутриигровых покупок от расы персонажа?
Согласно среднему количеству покупок на одного игрока, раса “Human” в среднем делает 121.4 покупку, тогда как раса “ Demon” в среднем делает 77.87 покупку на игрока. Дисбаланс на более чем 50 % между расами. Вывод раса “Human” делает покупки чаще чем раса “ Demon” на 54%.
Анализ средних расходов выявил существенную разницу между расами. Северяне ("Northman") тратят на 50% больше внутриигровой валюты на одну покупку (62 519,07 лепестков) по сравнению с расой “ Demon” (41 194,84 лепестков).
 Полученные данные подтверждают гипотезу о существенном дисбалансе в игровом опыте между расами. Различия затрагивают как частоту покупок (до +54%), так и их экономическую нагрузку (до +50% стоимости). Особенно выделяются расы "Human" (максимальная активность) и "Northman" (максимальные затраты), в то время как "Demon" показывает наименьшие значения по обоим показателям.
